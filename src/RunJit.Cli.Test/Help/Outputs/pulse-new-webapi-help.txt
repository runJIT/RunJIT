webapi:
  The command to create a new runjit web api solution

Usage:
  runjit new webapi [options]

Options:
  -o, --output <output> (REQUIRED)                The output directory where the generation should take place
  -pn, --project-name <projectName> (REQUIRED)    The name of the project. Sample: 'RunJit.Survey' or 'RunJit.DataManagement'
  -bp, --base-path <basePath> (REQUIRED)          The base path of your Api. Sample could be 'api/core', 'api/survey'
  -mn, --module-name <moduleName> (REQUIRED)      The name of the module. Sample: 'survey', 'core' or 'data-management'
